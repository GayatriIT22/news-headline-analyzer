# 📰 News Headline Analyzer

A Python command-line tool built during my internship at TechRBM.

## 🔧 Technologies Used
- Pure Python (No external libraries)
- Dictionaries, Sets, Lists, String Operations

## ✨ Features
- Classifies headlines into 9 categories
- Finds similar headlines
- Top 5 frequent words
- Search by keyword
- Categorize your own headline
- Most covered topic

## 🚀 How to Run
python news_headline_analyzer.py

## 🔮 Future Scope
- NewsAPI integration for live headlines
- Matplotlib charts for visualization
- NLP for smarter categorization
